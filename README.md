## Featured Projects

Autonomous Driving Perception  
Benchmarking CNN vs Vision Transformers using a multimodal dataset (~50k images + LiDAR). Explored image preprocessing (Sobel, HSV, grayscale) to analyze convergence and MAE/MSE performance.  
→ [Repository](https://github.com/Oso1718/CNN-vs-ViT-in-Autonomous-Driving-Analysis).

Spam Detection via Weighted Stacking  
Machine learning pipeline combining Random Forest, SVM, and Neural Networks into a weighted ensemble achieving ~96% accuracy and ~97% F1-score.  
→ [Repository](https://github.com/Oso1718/Spam-Detection-via-Weighted-Stacking-Ensembles).

Market Price Optimizer  
Multi-store grocery price optimization system using web scraping, PostgreSQL data pipelines, and product normalization. Designed to identify cost-efficient shopping strategies across retailers.  
→ [Repository](https://github.com/Oso1718/Multi-store-optimizer.).
